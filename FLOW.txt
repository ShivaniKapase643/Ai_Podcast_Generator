AI News Podcast Generator — How It Works
The Pipeline (What happens when you click "Generate Now")
1. SCRAPE → 2. RANK → 3. SUMMARIZE → 4. SCRIPT → 5. AUDIO → 6. SAVE → 7. DELIVER
Step 1: Scrape News — Pulls articles from 8 RSS feeds (TechCrunch, Wired, The Verge, VentureBeat, MIT Tech Review, Hacker News, ArXiv, ProductHunt). Falls back to Bright Data proxy scraping if RSS fails.

Step 2: AI Ranking — Sends all articles to GPT-4o, which picks the top 5-7 most important stories. Each gets tagged (#LLM, #Startup, #Research, etc.), a sentiment label, and a "why it matters" blurb.

Step 3: Summarize — GPT-4o creates concise 2-3 sentence summaries of each selected story.

Step 4: Script Generation — GPT-4o writes a full podcast script (~1200 words, 8-10 min read) with intro, story segments, transitions, and outro. Conversational tone, timestamped.

Step 5: Text-to-Speech — ElevenLabs converts the script to a natural-sounding MP3 audio file.

Step 6: Save — Stores audio, transcript (.txt), PDF transcript, and metadata to the file system + SQLite database.

Step 7: Deliver — Sends email to all subscribers with audio link + PDF attachment.
